argonone-rs · UI mockups

Argon40 case monitor — web UI

Static mockups for the Rust daemon's web UI, covering the forced first-run setup through day-to-day fan/storage/display control. Each page is a standalone, self-contained HTML file — open any of them directly in a browser.

Screen flow

  • 1 First-run setup
    Forced admin account creation — only reachable route until it's done.
     unauthenticated
  • 2 Sign in
    Username + password — forgot it? Ask an admin, no email recovery.
     unauthenticated
  • 3 Dashboard
    Status strip + fan, network, storage, display, power and system summary cards.
     all roles
  • 4 Fan control
    Draggable temp→speed curve editor for CPU and HDD.
     operator+
  • 5 Storage & RAID
    Per-disk usage/temperature, array health.
     viewer+
  • 6 Display (OLED)
    Live panel preview, screen rotation, screensaver timing.
     operator+ · EON
  • 7 Users
    Admin/operator/viewer roles, invite flow, password resets.
     admin only
  • 8 System
    Units, RTC schedule, IR learning, HTTPS/remote access, firmware, danger zone.
     operator+
